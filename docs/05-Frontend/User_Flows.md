# User Flows

Bu belge, temel kullanıcı akışlarını tanımlar.

- QR kodla menü açma
- NFC doğrulama
- Garson çağırma
- Hesabı isteme
- Ödeme başlatma
- Rezervasyon yönetimi
- Ana sayfada marka arama
- Marka sayfasından şube seçme
- Platform admin ile mağaza oluşturma
- Mağaza oluşturunca otomatik route üretimi
- Üst admin ile tema özelleştirme
- Staff oluşturma ve rol atama
- Kampanya oluşturma ve zamanlama
- Popup kampanya yayına alma
- Şube admin üzerinden ürün ve kategori yönetimi
