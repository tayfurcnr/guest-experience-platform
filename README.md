# guest-experience-platform

Anlık Menü için ürün, tasarım ve teknik dokümantasyon ile birlikte demo arayüzünü içeren repository.

## Ne içeriyor

- Ürün gereksinimleri
- Release planları
- Mimari notlar
- API taslağı
- Frontend notları ve kullanıcı akışları
- Sprint dokümanları
- Next.js tabanlı demo arayüzü

## Proje yapısı

- `app/` - Next.js demo arayüzü
- `docs/` - PRD, release, mimari ve frontend dokümanları
- `anlik-menu-prototype.html` - eski prototip referansı

## Çalıştırma

```bash
npm install
npm run dev
```

Uygulama varsayılan olarak `http://localhost:3000` adresinde açılır.

## Not

Bu repository şu anda ağırlıklı olarak ürün keşif ve demo arayüz çalışmaları için kullanılıyor.
